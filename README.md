# HeroComputerCRM



API CRM для управления досками, задачами и уведомлениями (Nuxt 3 + Prisma + MySQL).

---

## Установка и запуск

Скопируйте и выполните команды в терминале:

```bash
git clone https://github.com/Tudovchanin/HeroComputerCRM.git
cd HeroComputerCRM
npm install
cp -n .env.example .env
npm run migrate:dev
npm run dev
